# 验收与故障排查

## 已通过的HOBJ验收

正常样本`normal_10_16.hobj`：

```text
job_status=completed
measurement_status=success
连续实测约11～12秒
```

异常样本`abnormal_10_33.hobj`：

```text
job_status=completed
measurement_status=partial_success
AB/CD/AD/CB四个独立主面角保留
CD、CB倒角不可用
依赖CD、CB端点的对角线不可用
```

异常样本实测约26.55秒，超过20秒目标，后续仍需针对异常路径优化；功能上不影响正常
字段保留和错误原因返回。

## 现场验收清单

1. 解压到全新空目录；
2. 校验ZIP SHA256和`SHA256SUMS.txt`；
3. 安装服务并确认`ready=true`；
4. 正常HOBJ连续测量3次；
5. 异常HOBJ验证`partial_success`；
6. 计算中终止Worker，验证自动恢复；
7. 提交一个运行任务、一个等待任务，第三个请求应返回429；
8. 使用现场四TIFF验证格式；
9. 同源HOBJ/TIFF全部正式数值和可用性逐字段对比；
10. 在目标生产机执行连续样本20秒性能验收。

## 常见问题

### 无法连接127.0.0.1:8780

检查Windows服务、8780端口、`service_logs`和Windows事件查看器。

### ready=false

Worker仍在导入算法或模型校验失败。查看`fault_message`和算法日志，不要提交任务。

### HTTP 429

一个任务正在计算且等待队列已有一个任务。C#提示排队并稍后重试。

### partial_success

这是有效业务结果。显示所有`available`项目，并把`unavailable`项目显示为“不可用+
原因”，不得将整单当成程序崩溃。

### 计算超过20秒

服务返回`CALCULATION_SLOW`告警但继续计算。60秒是Worker卡死保护线，不等于正常性能
指标。保存`job_id`并检查审计结果中的分阶段耗时。

### 服务错误1067

检查配置路径、程序目录是否完整、服务账户权限，以及WinSW和算法服务日志。
